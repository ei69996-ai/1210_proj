# 이미지 최적화 수동 테스트 체크리스트

> 이미지 최적화 작업 완료 후 서버 실행하여 확인해야 할 항목들

## 📋 테스트 전 준비사항

1. 개발 서버 실행: `pnpm dev`
2. 브라우저 개발자 도구 열기 (F12)
   - Network 탭 열어두기 (이미지 로딩 확인용)
   - Console 탭 열어두기 (에러 확인용)

---

## 1️⃣ 홈페이지 (/) - 관광지 카드 이미지 확인

### 1.1 기본 이미지 로딩 확인
- [ ] **홈페이지 접속** (`http://localhost:3000`)
- [ ] **관광지 카드 이미지가 정상적으로 표시되는지 확인**
  - 이미지가 있는 카드: 실제 관광지 이미지가 보여야 함
  - 이미지가 없는 카드: 회색 배경에 이미지 아이콘이 보여야 함
- [ ] **에러가 없는지 확인**
  - 브라우저 Console에 빨간색 에러 메시지가 없어야 함
  - 이미지가 깨지거나 X 표시가 나오지 않아야 함

### 1.2 Priority 이미지 확인 (첫 6개 카드)
- [ ] **페이지 새로고침 (F5)**
- [ ] **Network 탭에서 이미지 로딩 순서 확인**
  - 첫 6개 카드의 이미지가 먼저 로딩되는지 확인
  - 나머지 카드 이미지는 스크롤할 때 로딩되는지 확인

### 1.3 Lazy Loading 확인
- [ ] **페이지를 아래로 스크롤**
- [ ] **Network 탭에서 확인**
  - 스크롤하면서 새로운 이미지가 로딩되는지 확인
  - 화면에 보이지 않는 이미지는 로딩되지 않는지 확인

### 1.4 이미지 품질 확인
- [ ] **카드 이미지 클릭하여 확대해서 보기**
- [ ] **이미지가 선명하게 보이는지 확인**
  - 너무 흐리지 않아야 함
  - 너무 무겁지 않아야 함 (로딩 속도)

### 1.5 반응형 이미지 확인
- [ ] **브라우저 창 크기 조절** (모바일, 태블릿, 데스크톱)
- [ ] **각 화면 크기에서 이미지가 적절한 크기로 표시되는지 확인**
  - 모바일: 이미지가 화면 너비에 맞게 표시
  - 데스크톱: 이미지가 적절한 크기로 표시

## 2️⃣ 상세페이지 (/places/[contentId]) - 기본 정보 이미지 확인

### 2.1 관광지 카드 클릭하여 상세페이지 이동
- [ ] **홈페이지에서 관광지 카드 하나 클릭**
- [ ] **상세페이지로 이동하는지 확인**

### 2.2 기본 정보 섹션 - 대표 이미지 확인
- [ ] **상단에 큰 대표 이미지가 표시되는지 확인**
  - 이미지가 있으면: 실제 관광지 이미지 표시
  - 이미지가 없으면: 회색 배경에 이미지 아이콘 표시
- [ ] **이미지가 빠르게 로딩되는지 확인** (priority 이미지)
- [ ] **이미지 품질이 선명한지 확인** (quality=90)

### 2.3 이미지 에러 처리 확인
- [ ] **이미지가 없는 관광지 상세페이지도 확인**
  - 에러 없이 플레이스홀더가 표시되는지 확인
  - 페이지가 정상적으로 로드되는지 확인

---

## 3️⃣ 상세페이지 - 이미지 갤러리 확인

### 3.1 이미지 갤러리 기본 확인
- [ ] **이미지 갤러리 섹션이 있는지 확인**
  - 이미지가 있는 관광지: 갤러리 섹션 표시
  - 이미지가 없는 관광지: 갤러리 섹션 숨김

### 3.2 첫 번째 이미지 Priority 확인
- [ ] **갤러리의 첫 번째 이미지가 먼저 로딩되는지 확인**
  - Network 탭에서 첫 번째 이미지가 우선 로딩되는지 확인

### 3.3 이미지 슬라이드 확인
- [ ] **갤러리에서 좌우 화살표 버튼 클릭**
- [ ] **이미지가 슬라이드로 넘어가는지 확인**
- [ ] **나머지 이미지들이 lazy loading으로 로딩되는지 확인**

### 3.4 썸네일 그리드 확인
- [ ] **갤러리 아래 썸네일 그리드가 표시되는지 확인**
- [ ] **썸네일 이미지가 적절한 품질로 표시되는지 확인** (quality=75)
- [ ] **썸네일 클릭 시 메인 이미지가 변경되는지 확인**

### 3.5 전체화면 모달 확인
- [ ] **갤러리의 메인 이미지 클릭**
- [ ] **전체화면 모달이 열리는지 확인**
- [ ] **모달 내에서 이미지가 선명하게 표시되는지 확인** (quality=95)
- [ ] **모달 내에서 좌우 화살표로 이미지 이동 가능한지 확인**
- [ ] **ESC 키 또는 X 버튼으로 모달 닫기 가능한지 확인**

## 5️⃣ 다양한 시나리오 확인

### 5.1 이미지가 많은 관광지
- [ ] **이미지가 많은 관광지 상세페이지 확인**
- [ ] **모든 이미지가 정상적으로 로딩되는지 확인**
- [ ] **페이지 성능이 느려지지 않는지 확인**

### 5.2 이미지가 없는 관광지
- [ ] **이미지가 전혀 없는 관광지 확인**
- [ ] **플레이스홀더가 적절하게 표시되는지 확인**
- [ ] **에러 없이 페이지가 로드되는지 확인**

### 5.3 네트워크 속도 테스트

## 6️⃣ 에러 확인

- [ ] **빨간색 에러 메시지가 없는지 확인**

### 6.2 Network 에러 확인
- [ ] **Network 탭에서 이미지 요청 확인**
- [ ] **실패한 이미지 요청(빨간색)이 없는지 확인**
