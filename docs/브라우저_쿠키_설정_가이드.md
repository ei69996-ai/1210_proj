# 브라우저 쿠키 설정 변경 가이드 (자세한 설명)

> 자물쇠 아이콘을 찾기 어려울 때 사용하는 가이드

---

## 🔍 자물쇠 아이콘 위치 설명

### Chrome/Edge 브라우저:

```
┌─────────────────────────────────────────────────────────┐
│ 🔒 localhost:3000                    [X] [-] [□]         │ ← 주소창
│    ↑ 여기!                                              │
│    이 자물쇠 아이콘을 클릭하세요                         │
└─────────────────────────────────────────────────────────┘
```

**위치**: 
- 브라우저 맨 위에 있는 주소창(URL을 입력하는 곳)의 **왼쪽 끝**
- `localhost:3000` 또는 `http://localhost:3000` 앞에 있는 🔒 아이콘

**만약 자물쇠 아이콘이 보이지 않는다면:**
- `http://`로 시작하는 경우: ⓘ (정보 아이콘) 또는 🔒 아이콘
- `https://`로 시작하는 경우: 🔒 (자물쇠 아이콘)
- 아이콘이 없을 수도 있습니다 (일부 브라우저)

---

## ✅ 방법 1: 주소창 아이콘 클릭 (가장 쉬움)

### Chrome/Edge:

1. **브라우저 맨 위 주소창을 찾으세요**
   - `localhost:3000` 또는 `http://localhost:3000`이 보이는 곳

2. **주소창 왼쪽 끝을 보세요**
   - 🔒 자물쇠 아이콘 또는 ⓘ 정보 아이콘이 있을 수 있습니다

3. **아이콘을 클릭하세요**
   - 클릭하면 팝업 창이 열립니다

4. **"쿠키 및 사이트 데이터" 또는 "Cookies" 클릭**

5. **"허용됨" 또는 "Allow"로 설정**

---

## ✅ 방법 2: 브라우저 설정에서 직접 변경 (자물쇠 아이콘 없을 때)

### Chrome 브라우저:

1. **브라우저 우측 상단의 점 3개(⋮) 아이콘 클릭**
   ```
   ┌─────────────────────────┐
   │  [⋮] ← 여기 클릭        │
   └─────────────────────────┘
   ```

2. **메뉴에서 "설정" 또는 "Settings" 클릭**

3. **왼쪽 메뉴에서 "개인정보 및 보안" 또는 "Privacy and security" 클릭**

4. **"쿠키 및 기타 사이트 데이터" 또는 "Cookies and other site data" 클릭**

5. **"모든 쿠키 허용" 또는 "Allow all cookies" 선택**

6. **또는 "차단된 쿠키 및 사이트 데이터"에서 localhost 추가**

---

### Edge 브라우저:

1. **브라우저 우측 상단의 점 3개(⋯) 아이콘 클릭**

2. **메뉴에서 "설정" 또는 "Settings" 클릭**

3. **왼쪽 메뉴에서 "쿠키 및 사이트 권한" 또는 "Cookies and site permissions" 클릭**

4. **"쿠키 및 사이트 데이터" 또는 "Cookies and site data" 클릭**

5. **"모든 쿠키 허용" 또는 "Allow all cookies" 선택**

---

### Firefox 브라우저:

1. **브라우저 우측 상단의 햄버거 메뉴(☰) 아이콘 클릭**

2. **메뉴에서 "설정" 또는 "Settings" 클릭**

3. **왼쪽 메뉴에서 "개인 정보 보호" 또는 "Privacy & Security" 클릭**

4. **"쿠키 및 사이트 데이터" 섹션 찾기**

5. **"모든 쿠키 허용" 또는 "Accept cookies from websites" 선택**

---

## ✅ 방법 3: 주소창에 직접 입력 (가장 확실함)

### Chrome/Edge:

1. **주소창에 다음을 입력하고 Enter:**
   ```
   chrome://settings/cookies
   ```
   또는 (Edge의 경우)
   ```
   edge://settings/cookies
   ```

2. **"모든 쿠키 허용" 선택**

---

## ✅ 방법 4: 사이트별로 허용하기

### Chrome/Edge:

1. **브라우저 설정 열기** (방법 2 참고)

2. **"쿠키 및 기타 사이트 데이터" 클릭**

3. **"차단된 쿠키 및 사이트 데이터" 또는 "Blocked" 섹션 찾기**

4. **"추가" 또는 "Add" 버튼 클릭**

5. **다음 입력:**
   ```
   localhost
   ```

6. **"추가" 클릭**

- 주소창이 숨겨져 있음

### 해결 방법:
- **방법 2 (브라우저 설정에서 직접 변경)** 사용

3. **웹사이트 접속** (`http://localhost:3000`)
