# Chrome 브라우저 쿠키 설정 가이드

> Chrome 브라우저에서 쿠키 설정을 변경하는 방법

---

## 🎯 방법 1: 주소창에 직접 입력 (가장 빠름, 추천!)

### 단계별 안내:

1. **Chrome 브라우저를 엽니다**

2. **주소창(URL 입력하는 곳)을 클릭합니다**
   ```
   ┌─────────────────────────────────────────┐
   │ localhost:3000                          │ ← 여기 클릭!
   └─────────────────────────────────────────┘
   ```

3. **다음을 입력합니다:**
   ```
   chrome://settings/cookies
   ```

4. **Enter 키를 누릅니다**

5. **설정 페이지가 열리면:**
   - "모든 쿠키 허용" 또는 "Allow all cookies" 선택
   - 또는 "차단된 쿠키 및 사이트 데이터" 섹션에서 "추가" 클릭
   - `localhost` 입력 후 "추가" 클릭

6. **Chrome 브라우저를 완전히 닫고 다시 엽니다**

---

## 🎯 방법 2: 메뉴에서 설정 열기

### 단계별 안내:

1. **Chrome 브라우저 우측 상단의 점 3개(⋮) 아이콘 클릭**
   ```
   ┌─────────────────────────────────────────┐
   │                              [⋮]       │ ← 여기!
   └─────────────────────────────────────────┘
   ```

2. **메뉴가 열리면 맨 아래 "설정" 클릭**
   ```
   ┌─────────────────────────────────────────┐
   │ 북마크                                  │
   │ 기록                                    │
   │ 다운로드                                │
   │ ...                                     │
   │ 설정 ← 여기!                            │
   └─────────────────────────────────────────┘
   ```

3. **설정 페이지 왼쪽 메뉴에서 "개인정보 및 보안" 클릭**
   ```
   ┌──────────┬─────────────────────────────┐
   │ 개인정보 │                              │
   │ 및 보안  │  ← 여기 클릭!                │
   └──────────┴─────────────────────────────┘
   ```

4. **"쿠키 및 기타 사이트 데이터" 클릭**

5. **"모든 쿠키 허용" 선택**
   ```
   ┌─────────────────────────────────────────┐
   │ 쿠키 및 기타 사이트 데이터               │
   │                                          │
   │ ○ 모든 쿠키 허용 ← 이거 선택!            │
   │ ○ 쿠키 차단                                │
   └─────────────────────────────────────────┘
   ```

6. **Chrome 브라우저를 완전히 닫고 다시 엽니다**

---

## 🎯 방법 3: 사이트별로 허용하기

### localhost만 허용하고 싶은 경우:

1. **주소창에 입력:**
   ```
   chrome://settings/cookies
   ```

2. **"차단된 쿠키 및 사이트 데이터" 섹션 찾기**

3. **"추가" 또는 "Add" 버튼 클릭**

4. **다음 입력:**
   ```
   localhost
   ```

5. **"추가" 클릭**

6. **Chrome 브라우저를 완전히 닫고 다시 엽니다**

---

## 🔍 자물쇠 아이콘 찾기 (선택 사항)

### Chrome에서 자물쇠 아이콘 위치:

1. **브라우저 맨 위 주소창을 보세요**
   ```
   ┌─────────────────────────────────────────┐
   │ 🔒 localhost:3000                       │
   │ ↑ 여기!                                  │
   └─────────────────────────────────────────┘
   ```

2. **자물쇠 아이콘(🔒)을 클릭**

3. **팝업 창에서 "쿠키 및 사이트 데이터" 클릭**

4. **"허용됨" 또는 "Allow" 확인**

**참고**: 
- `http://`로 시작하면 ⓘ (정보 아이콘)일 수 있습니다
- `https://`로 시작하면 🔒 (자물쇠 아이콘)입니다
- 아이콘이 보이지 않을 수도 있습니다

## ✅ 설정 후 확인 방법

1. **Chrome 브라우저를 완전히 닫기**
   - 모든 Chrome 창을 닫아야 합니다
   - 작업 표시줄의 Chrome 아이콘도 닫아야 합니다

2. **Chrome 브라우저 다시 열기**

3. **웹사이트 접속:**
   ```
   http://localhost:3000
   ```

4. **에러 확인:**
   - F12 키 누르기
   - Console 탭 클릭
   - 빨간색 에러 메시지가 사라졌는지 확인

## 💡 팁

### Chrome을 완전히 닫는 방법:

1. **모든 Chrome 창 닫기**
2. **작업 표시줄의 Chrome 아이콘 확인**
3. **아이콘에 마우스 오른쪽 클릭 → "종료" 또는 "Exit"**

또는

1. **작업 관리자 열기** (Ctrl + Shift + Esc)
2. **Chrome 프로세스 모두 종료**

## ❓ 문제 해결

### 설정이 저장되지 않는 경우:

1. **Chrome을 완전히 닫았는지 확인**
2. **Chrome을 다시 열고 설정 확인**
3. **여전히 안 되면 방법 1 (주소창 입력) 사용**

### 여전히 에러가 발생하는 경우:

1. **다른 브라우저(Edge, Firefox)에서 테스트**
2. **Chrome 확장 프로그램 비활성화** (광고 차단기 등)
3. **시크릿 모드가 아닌지 확인**
