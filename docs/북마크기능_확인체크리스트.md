# 북마크 기능 확인 체크리스트

> 개발된 북마크 기능을 직접 테스트하여 정상 동작 여부를 확인하는 체크리스트입니다.
> 서버를 실행한 후 브라우저에서 직접 확인하세요.

## 사전 준비

- [ ] 개발 서버 실행 (`pnpm dev`)
- [ ] 브라우저에서 `http://localhost:3000` 접속
- [ ] Supabase 연결 확인 (환경변수 설정 확인)

---

## 1. 기본 UI 확인

### 1.1 상세페이지 접속
- [ ] 홈페이지(`/`)에서 관광지 카드 하나 클릭
- [ ] 상세페이지(`/places/[contentId]`)로 이동 확인
- [ ] 페이지가 정상적으로 로드되는지 확인

### 1.2 북마크 버튼 표시 확인
- [ ] 상세페이지 헤더 영역(뒤로가기 버튼 오른쪽)에 북마크 버튼이 보이는지 확인
- [ ] 북마크 버튼이 "공유" 버튼 왼쪽에 배치되어 있는지 확인
- [ ] 북마크 버튼에 별 아이콘(⭐)이 표시되는지 확인
- [ ] 버튼에 "북마크" 텍스트가 표시되는지 확인

### 1.3 초기 로딩 상태 확인
- [ ] 페이지 로드 시 북마크 버튼에 로딩 스피너가 잠깐 표시되는지 확인
- [ ] 로딩이 완료되면 별 아이콘으로 변경되는지 확인

---

## 2. 로그인하지 않은 상태 테스트

### 2.1 로그아웃 상태 확인
- [ ] 현재 로그인하지 않은 상태인지 확인 (Navbar에 "로그인" 버튼이 보이는지 확인)

### 2.2 북마크 버튼 클릭 (로그인 안 함)
- [ ] 북마크 버튼 클릭
- [ ] 화면 하단 또는 상단에 토스트 알림이 나타나는지 확인
- [ ] 알림 메시지에 "북마크 기능을 사용하려면 로그인이 필요합니다"가 표시되는지 확인
- [ ] 알림에 "로그인" 버튼이 있는지 확인 (선택 사항)

### 2.3 별 아이콘 상태 확인
- [ ] 로그인하지 않은 상태에서는 별 아이콘이 비어있는 상태(빈 별)인지 확인

---

## 3. 로그인한 상태 테스트

### 3.1 로그인
- [ ] Navbar의 "로그인" 버튼 클릭
- [ ] Clerk 로그인 화면에서 로그인 완료
- [ ] 로그인 후 상세페이지로 돌아왔는지 확인

### 3.2 북마크 추가 테스트
- [ ] 북마크 버튼 클릭 (별 아이콘이 비어있는 상태)
- [ ] 버튼 클릭 시 로딩 스피너가 잠깐 표시되는지 확인
- [ ] 로딩 완료 후 별 아이콘이 채워진 상태(노란색 별 ⭐)로 변경되는지 확인
- [ ] 버튼 텍스트가 "북마크됨"으로 변경되는지 확인
- [ ] 화면에 토스트 알림이 나타나고 "북마크에 추가되었습니다" 메시지가 표시되는지 확인

### 3.3 북마크 제거 테스트
- [ ] 북마크가 추가된 상태에서 북마크 버튼 다시 클릭
- [ ] 버튼 클릭 시 로딩 스피너가 잠깐 표시되는지 확인
- [ ] 로딩 완료 후 별 아이콘이 비어있는 상태로 변경되는지 확인
- [ ] 버튼 텍스트가 "북마크"로 변경되는지 확인
- [ ] 화면에 토스트 알림이 나타나고 "북마크에서 제거되었습니다" 메시지가 표시되는지 확인

### 3.4 북마크 상태 유지 확인
- [ ] 북마크를 추가한 상태에서 페이지 새로고침(F5 또는 Ctrl+R)
- [ ] 새로고침 후에도 별 아이콘이 채워진 상태로 유지되는지 확인
- [ ] 버튼 텍스트가 "북마크됨"으로 유지되는지 확인

---

## 4. 여러 관광지 테스트

### 4.1 다른 관광지 북마크
- [ ] 홈으로 돌아가기
- [ ] 다른 관광지 카드 클릭하여 상세페이지 이동
- [ ] 새 관광지에서 북마크 버튼 클릭하여 북마크 추가
- [ ] 북마크가 정상적으로 추가되는지 확인

### 4.2 북마크 상태 독립성 확인
- [ ] 첫 번째 관광지 상세페이지로 다시 이동
- [ ] 첫 번째 관광지의 북마크 상태가 유지되는지 확인
- [ ] 두 번째 관광지 상세페이지로 이동
- [ ] 두 번째 관광지의 북마크 상태가 유지되는지 확인

---

## 5. 반응형 디자인 확인

### 5.1 모바일 화면 (선택 사항)
- [ ] 브라우저 개발자 도구 열기 (F12)
- [ ] 모바일 화면 모드로 전환 (예: iPhone 12 Pro)
- [ ] 상세페이지 접속
- [ ] 북마크 버튼이 정상적으로 보이는지 확인
- [ ] 버튼 클릭이 정상적으로 동작하는지 확인

### 5.2 태블릿 화면 (선택 사항)
- [ ] 태블릿 화면 크기로 조정
- [ ] 북마크 버튼이 정상적으로 보이는지 확인

---

## 6. 에러 처리 확인 (선택 사항)

### 6.1 네트워크 에러 시뮬레이션
- [ ] 브라우저 개발자 도구 열기 (F12)
- [ ] Network 탭에서 "Offline" 모드로 전환
- [ ] 북마크 버튼 클릭
- [ ] 에러 메시지가 표시되는지 확인

### 6.2 빠른 연속 클릭 테스트
- [ ] 북마크 버튼을 빠르게 여러 번 클릭
- [ ] 중복 요청이 발생하지 않는지 확인 (로딩 중에는 버튼이 비활성화되어야 함)

---

## 7. 접근성 확인 (선택 사항)

### 7.1 키보드 네비게이션
- [ ] Tab 키를 눌러 북마크 버튼에 포커스 이동
- [ ] Enter 키 또는 Space 키로 북마크 버튼 클릭
- [ ] 키보드로도 정상 동작하는지 확인

### 7.2 스크린 리더 (선택 사항)
- [ ] 스크린 리더 활성화 (Windows: Narrator, Mac: VoiceOver)
- [ ] 북마크 버튼에 포커스 이동
- [ ] "북마크 추가" 또는 "북마크 제거"가 읽히는지 확인

---

## 빠른 확인 요약 체크리스트

> 시간이 부족할 때 빠르게 확인할 핵심 사항만 체크하세요.

### 필수 확인 사항 (5분)
- [ ] 상세페이지에 북마크 버튼이 보이는가?
- [ ] 로그인하지 않은 상태에서 북마크 버튼 클릭 시 로그인 유도 메시지가 나타나는가?
- [ ] 로그인한 상태에서 북마크 버튼 클릭 시 별 아이콘이 채워지는가?
- [ ] 북마크 추가 후 페이지 새로고침해도 북마크 상태가 유지되는가?
- [ ] 북마크 제거 시 별 아이콘이 비어지고 토스트 알림이 나타나는가?

### 추가 확인 사항 (10분)
- [ ] 여러 관광지를 북마크하고 각각의 상태가 독립적으로 유지되는가?
- [ ] 북마크 추가/제거 시 토스트 알림이 정상적으로 표시되는가?
- [ ] 로딩 상태가 적절하게 표시되는가?
- [ ] 버튼이 로딩 중일 때 중복 클릭이 방지되는가?
