# 이미지 갤러리 기능 수동 테스트 체크리스트

> 개발 서버 실행 후 실제로 클릭하고 확인하는 테스트 항목들입니다.

## 📋 테스트 전 준비사항

1. **개발 서버 실행**
   ```bash
   pnpm dev
   ```
2. **브라우저에서 `http://localhost:3000` 접속**

---

## ✅ 기본 확인 사항

### 1. 홈페이지에서 상세페이지 접근
- [ ] 홈페이지(`/`)에서 관광지 카드가 보이는지 확인
- [ ] 관광지 카드 중 하나를 클릭
- [ ] 상세페이지(`/places/[contentId]`)로 이동되는지 확인
- [ ] 상세페이지에 "뒤로가기" 버튼이 보이는지 확인

### 2. 이미지 갤러리 섹션 표시 확인
- [ ] 상세페이지에서 "기본 정보" 섹션 아래에 "이미지 갤러리" 섹션이 보이는지 확인
- [ ] "이미지 갤러리" 제목이 보이는지 확인
- [ ] "총 X개의 이미지" 텍스트가 보이는지 확인 (X는 숫자)

**참고**: 이미지가 없는 관광지의 경우 갤러리 섹션이 아예 안 보일 수 있습니다. 이는 정상입니다.

---

## 🖼️ 이미지가 있는 경우 테스트

### 3. 메인 갤러리 (큰 이미지 슬라이드)
- [ ] 큰 이미지가 하나 보이는지 확인
- [ ] 이미지가 2개 이상인 경우:
  - [ ] 이미지 왼쪽에 "◀" 버튼이 보이는지 확인
  - [ ] 이미지 오른쪽에 "▶" 버튼이 보이는지 확인
  - [ ] 이미지 아래에 작은 점들(페이지네이션)이 보이는지 확인
  - [ ] "◀" 버튼을 클릭했을 때 이전 이미지로 넘어가는지 확인
  - [ ] "▶" 버튼을 클릭했을 때 다음 이미지로 넘어가는지 확인
  - [ ] 작은 점들을 클릭했을 때 해당 이미지로 이동하는지 확인

### 4. 썸네일 그리드 (작은 이미지들)
- [ ] 큰 이미지 아래에 작은 이미지들이 4개씩 줄지어 보이는지 확인 (모바일 기준)
- [ ] 썸네일 이미지에 마우스를 올렸을 때 약간 커지는지 확인 (호버 효과)
- [ ] 썸네일 이미지를 클릭했을 때 큰 이미지가 해당 이미지로 바뀌는지 확인
- [ ] 현재 선택된 썸네일이 테두리로 강조되는지 확인

### 5. 전체화면 모달 테스트
- [ ] 큰 이미지를 클릭했을 때 전체화면 모달이 열리는지 확인
- [ ] 모달이 열렸을 때:
  - [ ] 화면 전체가 어두워지는지 확인 (배경 오버레이)
  - [ ] 이미지가 화면 중앙에 크게 보이는지 확인
  - [ ] 모달 오른쪽 위에 "X" 닫기 버튼이 보이는지 확인
  - [ ] 이미지 왼쪽에 "◀" 버튼이 보이는지 확인
  - [ ] 이미지 오른쪽에 "▶" 버튼이 보이는지 확인
  - [ ] 이미지 아래 중앙에 "1 / 5" 같은 인덱스 표시가 보이는지 확인

### 6. 모달 내 네비게이션 테스트
- [ ] 모달이 열린 상태에서:
  - [ ] "◀" 버튼을 클릭했을 때 이전 이미지로 넘어가는지 확인
  - [ ] "▶" 버튼을 클릭했을 때 다음 이미지로 넘어가는지 확인
  - [ ] 이미지 아래 작은 점들을 클릭했을 때 해당 이미지로 이동하는지 확인
  - [ ] 인덱스 숫자가 이미지 변경에 따라 업데이트되는지 확인

### 7. 모달 닫기 테스트
- [ ] 모달이 열린 상태에서:
  - [ ] "X" 버튼을 클릭했을 때 모달이 닫히는지 확인
  - [ ] 모달 배경(어두운 부분)을 클릭했을 때 모달이 닫히는지 확인 (Dialog 컴포넌트 기본 동작)

### 8. 키보드 네비게이션 테스트
- [ ] 모달이 열린 상태에서:
  - [ ] 키보드 `ESC` 키를 눌렀을 때 모달이 닫히는지 확인
  - [ ] 키보드 `←` (왼쪽 화살표) 키를 눌렀을 때 이전 이미지로 넘어가는지 확인
  - [ ] 키보드 `→` (오른쪽 화살표) 키를 눌렀을 때 다음 이미지로 넘어가는지 확인
  - [ ] 모달이 닫힌 상태에서 메인 갤러리 이미지에 포커스가 있을 때:
    - [ ] `Enter` 또는 `Space` 키를 눌렀을 때 모달이 열리는지 확인

## 📱 반응형 디자인 테스트

### 9. 모바일 화면 테스트 (브라우저 개발자 도구 사용)
- [ ] 브라우저 개발자 도구 열기 (F12 또는 우클릭 → 검사)
- [ ] 디바이스 모드로 전환 (Ctrl+Shift+M 또는 개발자 도구 상단 아이콘)
- [ ] 모바일 화면 크기로 조정 (예: iPhone 12 Pro, 390x844)
- [ ] 썸네일 그리드가 4개씩 줄지어 보이는지 확인
- [ ] 모달이 열렸을 때 화면 전체를 잘 채우는지 확인
- [ ] 모달 내 네비게이션 버튼이 잘 보이는지 확인

### 10. 태블릿 화면 테스트
- [ ] 태블릿 화면 크기로 조정 (예: iPad, 768x1024)
- [ ] 썸네일 그리드가 6개씩 줄지어 보이는지 확인
- [ ] 이미지와 버튼들이 적절한 크기로 보이는지 확인

### 11. 데스크톱 화면 테스트
- [ ] 데스크톱 화면 크기로 조정 (예: 1920x1080)
- [ ] 썸네일 그리드가 8개씩 줄지어 보이는지 확인
- [ ] 이미지가 적절한 크기로 보이는지 확인

## 🚫 에러 처리 테스트

### 12. 이미지가 없는 관광지 테스트
- [ ] 이미지가 없는 관광지 상세페이지로 이동
- [ ] "이미지 갤러리" 섹션이 아예 안 보이는지 확인 (정상 동작)
- [ ] 다른 섹션들(기본 정보, 운영 정보)은 정상적으로 보이는지 확인

### 13. 이미지 로딩 실패 테스트
- [ ] 이미지가 있는 관광지 상세페이지로 이동
- [ ] 네트워크 탭에서 특정 이미지 요청을 차단 (개발자 도구 → Network → 특정 이미지 우클릭 → Block request URL)
- [ ] 페이지를 새로고침
- [ ] 이미지 로딩 실패 시 플레이스홀더 아이콘이 보이는지 확인 (카메라 모양 아이콘)

---

## 🔄 페이지 새로고침 및 재접근 테스트

### 14. 페이지 새로고침
- [ ] 상세페이지에서 F5 또는 새로고침 버튼 클릭
- [ ] 이미지 갤러리가 정상적으로 로드되는지 확인
- [ ] 모든 기능이 정상 작동하는지 확인

### 15. 다른 관광지로 이동
- [ ] "뒤로가기" 버튼을 클릭하여 홈으로 이동
- [ ] 다른 관광지 카드를 클릭
- [ ] 새로운 관광지의 이미지 갤러리가 정상적으로 표시되는지 확인

## ⚡ 성능 및 사용성 테스트

### 16. 이미지 로딩 속도
- [ ] 이미지가 로드될 때까지 시간이 적절한지 확인 (3초 이내 권장)
- [ ] 이미지가 로드되는 동안 깜빡임이나 레이아웃 이동이 없는지 확인

### 17. 부드러운 전환 효과
- [ ] 이미지 슬라이드 전환 시 부드럽게 넘어가는지 확인
- [ ] 모달 열기/닫기 시 애니메이션이 부드러운지 확인
- [ ] 썸네일 호버 효과가 자연스러운지 확인

### 18. 여러 이미지가 있는 경우
- [ ] 이미지가 8개 이상인 관광지를 찾아서 테스트
- [ ] 썸네일 그리드에 "+X개의 이미지 더 보기" 텍스트가 보이는지 확인
- [ ] 메인 갤러리에서 모든 이미지를 슬라이드로 볼 수 있는지 확인

## 🎯 빠른 확인 체크사항 (요약)

다음 항목들만 빠르게 확인해도 기본 기능이 정상 작동하는지 알 수 있습니다:

### 필수 확인 (5분)
- [ ] ✅ 홈페이지에서 관광지 카드 클릭 → 상세페이지 이동
- [ ] ✅ 상세페이지에 "이미지 갤러리" 섹션이 보이는지 확인
- [ ] ✅ 큰 이미지를 클릭 → 전체화면 모달이 열리는지 확인
- [ ] ✅ 모달에서 "◀" "▶" 버튼으로 이미지 이동이 되는지 확인
- [ ] ✅ 모달에서 "X" 버튼 또는 ESC 키로 모달이 닫히는지 확인

### 추가 확인 (10분)
- [ ] ✅ 썸네일 이미지를 클릭했을 때 큰 이미지가 바뀌는지 확인
- [ ] ✅ 키보드 화살표 키(← →)로 이미지 이동이 되는지 확인
- [ ] ✅ 모바일 화면에서도 정상적으로 보이는지 확인 (F12 → 디바이스 모드)
- [ ] ✅ 이미지가 없는 관광지에서 갤러리 섹션이 안 보이는지 확인 (정상)

### 선택 확인 (시간 있을 때)
- [ ] ✅ 여러 이미지가 있는 관광지에서 모든 이미지를 슬라이드로 볼 수 있는지 확인
- [ ] ✅ 페이지 새로고침 후에도 정상 작동하는지 확인
- [ ] ✅ 다른 관광지로 이동 후 갤러리가 정상 표시되는지 확인

## 💡 참고사항

- **이미지가 없는 관광지**: 일부 관광지는 이미지가 없을 수 있습니다. 이 경우 갤러리 섹션이 아예 안 보이는 것이 정상입니다.
- **이미지 로딩 시간**: 한국관광공사 API 서버 상태에 따라 이미지 로딩 시간이 다를 수 있습니다.
- **브라우저 호환성**: Chrome, Firefox, Safari, Edge 등 주요 브라우저에서 테스트하는 것을 권장합니다.
- **모바일 테스트**: 실제 모바일 기기에서도 테스트하는 것을 권장합니다 (개발자 도구의 디바이스 모드와는 다를 수 있음).
