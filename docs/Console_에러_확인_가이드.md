# 브라우저 Console에서 에러 확인하는 방법

> 비개발자를 위한 쉬운 가이드

## 🎯 목적

개발된 웹사이트에서 발생하는 에러나 경고 메시지를 확인하기 위해 브라우저의 개발자 도구 Console을 사용합니다.

---

## 📖 단계별 가이드

### 방법 1: 키보드 단축키 사용 (가장 빠름)

1. **웹사이트를 열어둔 상태에서**
2. **키보드에서 `F12` 키를 누릅니다**
   - 또는 `Ctrl + Shift + I` (Windows/Linux)
   - 또는 `Cmd + Option + I` (Mac)
3. **개발자 도구가 열립니다**
4. **상단에 있는 탭 중에서 `Console` 탭을 클릭합니다**

### 방법 2: 마우스로 열기

1. **웹사이트 화면에서 빈 공간에 마우스 오른쪽 버튼 클릭**
2. **메뉴에서 "검사" 또는 "Inspect" 클릭**
   - 또는 "요소 검사" / "Inspect Element"
3. **개발자 도구가 열립니다**
4. **상단에 있는 탭 중에서 `Console` 탭을 클릭합니다**

---

## 🔍 Console 화면 설명

Console 탭이 열리면 다음과 같은 화면을 볼 수 있습니다:

```
┌─────────────────────────────────────────┐
│ Elements  Console  Sources  Network ... │ ← 탭 메뉴
├─────────────────────────────────────────┤
│                                         │
│  (여기에 메시지들이 표시됩니다)          │
│                                         │
│  ✅ 정상 메시지 (흰색 또는 회색)         │
│  ⚠️  경고 메시지 (노란색)                │
│  ❌ 에러 메시지 (빨간색)                 │
│                                         │
└─────────────────────────────────────────┘
```

---

## ✅ 확인해야 할 것들

### 1. 에러 메시지 확인 (가장 중요!)

- **빨간색 메시지가 있는지 확인**
  - 빨간색 = 에러 (문제가 있음)
  - 빨간색 메시지가 있으면 개발자에게 알려주세요

### 2. 경고 메시지 확인

- **노란색 메시지가 있는지 확인**
  - 노란색 = 경고 (심각하지 않지만 개선 가능)
  - 노란색 메시지는 있어도 괜찮지만, 많으면 알려주세요

### 3. 정상 메시지

- **흰색 또는 회색 메시지**
  - 정상적인 로그 메시지입니다
  - 무시해도 됩니다

## 📸 실제 예시

### ✅ 정상적인 경우 (에러 없음)

```
Console
├─ (빨간색 메시지 없음)
├─ (노란색 메시지 없거나 적음)
└─ 흰색/회색 메시지만 있음
```

**결과**: ✅ 정상 작동 중

### ❌ 문제가 있는 경우 (에러 있음)

```
Console
├─ ❌ Error: Invalid src prop...
├─ ❌ TypeError: Cannot read property...
└─ ⚠️  Warning: Image failed to load...
```

**결과**: ❌ 문제 발견 - 개발자에게 알려주세요

## 🧹 Console 초기화하기

Console에 메시지가 너무 많아서 확인하기 어려울 때:

1. **Console 탭이 열려있는 상태에서**
2. **Console 화면 왼쪽 상단에 있는 🚫 아이콘 클릭**
   - 또는 `Ctrl + L` (Windows/Linux)
   - 또는 `Cmd + K` (Mac)
3. **모든 메시지가 지워집니다**
4. **페이지를 새로고침 (F5)하여 새로 확인**

---

## 🔄 페이지 새로고침 후 확인

1. **Console 탭을 열어둔 상태에서**
2. **페이지를 새로고침 (F5)**
3. **새로고침 후 Console에 나타나는 메시지 확인**
   - 특히 빨간색 에러 메시지가 있는지 확인

---

## 📝 에러 발견 시 기록할 내용

에러를 발견했다면 다음 정보를 기록해주세요:

1. **에러 메시지 전체 내용** (빨간색 텍스트)
2. **어떤 페이지에서 발생했는지** (홈페이지 / 상세페이지)
3. **어떤 동작을 했을 때 발생했는지** (페이지 로드 / 이미지 클릭 등)
4. **스크린샷** (가능하면)

## 💡 팁

### Console 탭을 항상 열어두기

- 개발 중에는 Console 탭을 항상 열어두면 에러를 즉시 확인할 수 있습니다
- 개발자 도구는 화면 하단이나 오른쪽에 고정할 수 있습니다

### 에러 메시지 클릭하기

- 에러 메시지를 클릭하면 어디서 발생했는지 자세히 볼 수 있습니다
- 개발자에게 전달할 때 유용합니다

### 필터 기능 사용하기

- Console 상단에 필터 입력창이 있습니다
- "error" 또는 "에러"를 입력하면 에러만 볼 수 있습니다
