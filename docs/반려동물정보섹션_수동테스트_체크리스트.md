# 반려동물 정보 섹션 수동 테스트 체크리스트

> 반려동물 정보 섹션(MVP 2.5) 개발 완료 후 수동 테스트 가이드
> 
> **테스트 방법**: 서버 실행 후 브라우저에서 직접 확인

---

## 📋 테스트 전 준비사항

1. **서버 실행**
   ```bash
   pnpm dev
   ```
   - 서버가 정상적으로 실행되는지 확인
   - 브라우저에서 `http://localhost:3000` 접속 확인

2. **테스트용 관광지 찾기**
   - 홈페이지(`/`)에서 관광지 목록 확인
   - 관광지 카드를 클릭하여 상세페이지로 이동
   - 또는 직접 URL 입력: `/places/[contentId]` (예: `/places/125266`)

---

## ✅ 체크리스트

### 1. 기본 접근 및 섹션 표시 확인

#### 1.1 상세페이지 접근
- [ ] 홈페이지에서 관광지 카드를 클릭하면 상세페이지로 이동하는지 확인
- [ ] 상세페이지 URL이 `/places/[숫자]` 형태인지 확인
- [ ] 페이지가 정상적으로 로드되는지 확인 (에러 메시지 없음)

#### 1.2 반려동물 정보 섹션 위치 확인
- [ ] 상세페이지에서 아래 순서로 섹션이 배치되어 있는지 확인:
  1. 기본 정보 섹션 (관광지명, 이미지, 주소 등)
  2. 이미지 갤러리 섹션 (있는 경우)
  3. 지도 섹션 (있는 경우)
  4. **반려동물 정보 섹션** ← 새로 추가된 섹션
  5. 운영 정보 섹션

#### 1.3 반려동물 정보 섹션 표시 여부
- [ ] 반려동물 동반 가능한 관광지의 경우 "반려동물 정보" 섹션이 표시되는지 확인
- [ ] 반려동물 동반 불가능한 관광지의 경우 섹션이 숨겨지는지 확인
- [ ] 반려동물 정보가 없는 관광지의 경우 섹션이 숨겨지는지 확인

---

### 2. 섹션 제목 및 뱃지 확인

#### 2.1 섹션 제목
- [ ] "반려동물 정보" 제목이 큰 글씨로 표시되는지 확인
- [ ] 제목 옆에 "🐾 동반 가능" 뱃지가 표시되는지 확인
- [ ] 뱃지 색상이 눈에 잘 띄는지 확인 (연한 파란색 배경)

---

### 3. 반려동물 정보 필드 확인

#### 3.1 반려동물 크기 제한 정보
- [ ] "반려동물 크기" 항목이 표시되는지 확인
- [ ] 크기 정보가 뱃지 형태로 표시되는지 확인
  - 소형견: 파란색 뱃지
  - 중형견: 초록색 뱃지
  - 대형견: 주황색 뱃지
- [ ] 여러 크기가 있는 경우 (예: "소형, 중형") 모두 표시되는지 확인
- [ ] 각 뱃지에 🐾 아이콘이 있는지 확인

#### 3.2 입장 가능 장소
- [ ] "입장 가능 장소" 항목이 표시되는지 확인
- [ ] 아이콘이 표시되는지 확인:
  - 실내: 집 아이콘 (🏠)
  - 실외: 나무 아이콘 (🌲)
  - 모두: 집 + 나무 아이콘 둘 다
- [ ] 텍스트가 명확하게 표시되는지 확인 (예: "실내 입장 가능", "실외 입장 가능", "실내/실외 모두 입장 가능")

#### 3.3 추가 요금 정보
- [ ] "추가 요금" 항목이 표시되는지 확인
- [ ] 요금 정보가 텍스트로 표시되는지 확인
- [ ] 💰 아이콘이 표시되는지 확인

---

### 4. 주의사항 섹션 확인

#### 4.1 주의사항 표시
- [ ] 반려동물 정보에 주의사항이 있는 경우 노란색 박스로 표시되는지 확인
- [ ] ⚠️ 경고 아이콘이 표시되는지 확인
- [ ] "주의사항" 제목이 굵은 글씨로 표시되는지 확인
- [ ] 주의사항 내용이 여러 줄로 표시되는지 확인 (줄바꿈 정상 작동)

#### 4.2 주의사항이 없는 경우
- [ ] 주의사항이 없는 관광지의 경우 노란색 박스가 표시되지 않는지 확인

---

### 5. 레이아웃 및 디자인 확인

#### 5.1 카드 스타일
- [ ] 반려동물 정보가 회색 테두리가 있는 카드 안에 표시되는지 확인
- [ ] 카드 배경색이 흰색(또는 다크모드에서 어두운 색)인지 확인
- [ ] 카드 안에 여백이 적절한지 확인

#### 5.2 그리드 레이아웃 (데스크톱)
- [ ] 데스크톱 화면에서 정보가 2열로 표시되는지 확인
- [ ] 각 항목이 아이콘 + 라벨 + 값 형태로 정렬되어 있는지 확인
- [ ] 항목 간 간격이 적절한지 확인

#### 5.3 모바일 레이아웃
- [ ] 모바일 화면(또는 브라우저 창을 좁게)에서 정보가 1열로 표시되는지 확인
- [ ] 모바일에서도 텍스트가 잘 보이는지 확인
- [ ] 뱃지와 아이콘이 모바일에서도 적절한 크기로 표시되는지 확인

---

### 6. 다양한 관광지 테스트

#### 6.1 반려동물 동반 가능한 관광지
- [ ] 여러 개의 반려동물 동반 가능 관광지를 확인
- [ ] 각 관광지마다 반려동물 정보가 제대로 표시되는지 확인
- [ ] 정보가 다른 관광지들도 모두 정상 표시되는지 확인

#### 6.2 반려동물 동반 불가능한 관광지
- [ ] 반려동물 동반 불가능한 관광지의 상세페이지 확인
- [ ] 반려동물 정보 섹션이 표시되지 않는지 확인
- [ ] 다른 섹션들(기본 정보, 운영 정보 등)은 정상 표시되는지 확인

#### 6.3 정보가 일부만 있는 관광지
- [ ] 크기 정보만 있고 장소 정보가 없는 경우
- [ ] 장소 정보만 있고 크기 정보가 없는 경우
- [ ] 추가 요금 정보만 있는 경우
- [ ] 각각의 경우에도 해당 정보만 정상 표시되는지 확인

---

### 7. 에러 처리 확인

#### 7.1 API 에러 처리
- [ ] 반려동물 정보 API 호출이 실패해도 페이지가 정상 표시되는지 확인
- [ ] 에러 메시지가 사용자에게 노출되지 않는지 확인
- [ ] 콘솔에 경고 메시지만 출력되는지 확인 (개발자 도구 F12)

#### 7.2 잘못된 contentId 처리
- [ ] 잘못된 관광지 ID로 접근 시 에러 메시지가 표시되는지 확인
- [ ] 에러 메시지가 사용자 친화적인지 확인

---

### 8. 접근성 확인 (선택 사항)

#### 8.1 키보드 네비게이션
- [ ] Tab 키로 모든 요소에 접근할 수 있는지 확인

#### 8.2 스크린 리더 (선택 사항)

## 🚀 빠른 확인 체크리스트 (5분 버전)

다음 항목만 빠르게 확인하면 됩니다:

### 필수 확인 사항
- [ ] **상세페이지 접근**: 홈에서 관광지 클릭 → 상세페이지 이동 확인
- [ ] **섹션 위치**: 지도 섹션 다음에 "반려동물 정보" 섹션이 있는지 확인
- [ ] **섹션 제목**: "반려동물 정보" 제목과 "🐾 동반 가능" 뱃지가 보이는지 확인
- [ ] **정보 표시**: 반려동물 크기, 입장 가능 장소, 추가 요금이 표시되는지 확인
- [ ] **주의사항**: 주의사항이 있으면 노란색 박스로 표시되는지 확인
- [ ] **반응형**: 브라우저 창 크기를 조절해도 레이아웃이 깨지지 않는지 확인
- [ ] **정보 없는 경우**: 반려동물 동반 불가능한 관광지에서 섹션이 숨겨지는지 확인

### 확인 방법
1. 서버 실행: `pnpm dev`
2. 브라우저에서 `http://localhost:3000` 접속
3. 홈페이지에서 관광지 카드 클릭
4. 상세페이지에서 "반려동물 정보" 섹션 확인
5. 여러 관광지를 확인하여 다양한 경우 테스트
