# 추천 관광지 섹션 확인 체크리스트

> 개발 완료된 추천 관광지 섹션을 수동으로 확인하는 체크리스트입니다.

## 사전 준비

1. 개발 서버 실행 확인
   ```bash
   pnpm dev
   ```
   - 브라우저에서 `http://localhost:3000` 접속 가능한지 확인

---

## 기본 확인 사항

### 1. 상세페이지 접근 확인

- [ ] 홈페이지(`/`)에서 관광지 카드 하나를 클릭
- [ ] 상세페이지(`/places/[contentId]`)로 이동되는지 확인
- [ ] 페이지가 정상적으로 로드되는지 확인

### 2. 추천 섹션 표시 확인

- [ ] 상세페이지를 아래로 스크롤
- [ ] 운영 정보 섹션 다음에 **"이런 관광지는 어떠세요?"** 제목이 보이는지 확인
- [ ] 제목 아래에 관광지 카드들이 표시되는지 확인

### 3. 추천 관광지 내용 확인

- [ ] 추천된 관광지 카드들이 보이는지 확인
- [ ] 각 카드에 다음 정보가 표시되는지 확인:
  - [ ] 썸네일 이미지 (또는 기본 이미지)
  - [ ] 관광지명
  - [ ] 주소
  - [ ] 관광 타입 뱃지 (예: "관광지", "문화시설" 등)

### 4. 현재 관광지 제외 확인

- [ ] 추천 섹션에 표시된 관광지들을 확인
- [ ] **현재 보고 있는 관광지가 추천 목록에 포함되지 않았는지** 확인
  - 예: "경복궁" 상세페이지를 보고 있다면, 추천 목록에 "경복궁"이 없어야 함

### 5. 추천 관광지 개수 확인

- [ ] 추천 관광지가 **최대 6개**로 제한되어 있는지 확인
- [ ] 6개보다 많지 않은지 확인

### 6. 카드 클릭 동작 확인

- [ ] 추천 섹션의 관광지 카드 하나를 클릭
- [ ] 해당 관광지의 상세페이지로 이동되는지 확인
- [ ] 새로 이동한 상세페이지도 정상적으로 표시되는지 확인

---

## 반응형 레이아웃 확인

### 7. 데스크톱 화면 확인 (브라우저 창 크게)

- [ ] 브라우저 창을 넓게 조정 (1200px 이상)
- [ ] 추천 관광지 카드가 **한 줄에 3개**씩 표시되는지 확인
- [ ] 카드 간격이 적절한지 확인

### 8. 태블릿 화면 확인 (브라우저 창 중간 크기)

- [ ] 브라우저 창을 중간 크기로 조정 (768px ~ 1200px)
- [ ] 추천 관광지 카드가 **한 줄에 2개**씩 표시되는지 확인
- [ ] 레이아웃이 깨지지 않는지 확인

### 9. 모바일 화면 확인 (브라우저 창 작게)

- [ ] 브라우저 창을 작게 조정 (768px 미만)
- [ ] 또는 브라우저 개발자 도구(F12)에서 모바일 뷰로 전환
- [ ] 추천 관광지 카드가 **한 줄에 1개**씩 표시되는지 확인
- [ ] 카드가 화면 너비에 맞게 표시되는지 확인

---

## 예외 상황 확인

### 10. 추천 관광지가 없는 경우

다음과 같은 경우를 테스트해보세요:

- [ ] 같은 지역 + 같은 타입의 다른 관광지가 없는 경우
  - 예: 특정 지역의 특정 타입 관광지가 1개만 있는 경우
- [ ] **"이런 관광지는 어떠세요?"** 섹션이 **표시되지 않는지** 확인
- [ ] 페이지는 정상적으로 표시되는지 확인 (에러 메시지가 나오지 않아야 함)

### 11. 지역 코드가 없는 관광지

- [ ] 지역 코드 정보가 없는 관광지 상세페이지 접근 시도
- [ ] 추천 섹션이 표시되지 않는지 확인
- [ ] 페이지는 정상적으로 표시되는지 확인

---

## 다양한 관광지 타입 확인

### 12. 관광 타입별 확인

다양한 타입의 관광지 상세페이지에서 추천 섹션을 확인해보세요:

- [ ] **관광지(12)** 타입 관광지 상세페이지
- [ ] **문화시설(14)** 타입 관광지 상세페이지
- [ ] **축제/행사(15)** 타입 관광지 상세페이지
- [ ] **음식점(39)** 타입 관광지 상세페이지
- [ ] 각 타입에서 추천 섹션이 정상적으로 표시되는지 확인

### 13. 지역별 확인

다양한 지역의 관광지 상세페이지에서 추천 섹션을 확인해보세요:

- [ ] **서울** 지역 관광지 상세페이지
- [ ] **부산** 지역 관광지 상세페이지
- [ ] **제주** 지역 관광지 상세페이지
- [ ] 각 지역에서 추천 섹션이 정상적으로 표시되는지 확인

---

## 섹션 배치 확인

### 14. 섹션 순서 확인

상세페이지를 위에서 아래로 스크롤하며 섹션 순서를 확인:

- [ ] **기본 정보 섹션** (관광지명, 이미지, 주소 등)
- [ ] **이미지 갤러리 섹션** (있는 경우)
- [ ] **지도 섹션** (있는 경우)
- [ ] **반려동물 정보 섹션** (있는 경우)
- [ ] **운영 정보 섹션** (운영시간, 휴무일 등)
- [ ] **추천 관광지 섹션** ← **운영 정보 섹션 다음에 위치하는지 확인**

---

## 빠른 확인 요약 (5분 체크)

아래 항목만 빠르게 확인하면 기본 동작을 검증할 수 있습니다:

### 필수 확인 사항 (5분)

1. **상세페이지 접근**
   - [ ] 홈에서 관광지 카드 클릭 → 상세페이지 이동 확인

2. **추천 섹션 표시**
   - [ ] 페이지 아래로 스크롤 → **"이런 관광지는 어떠세요?"** 제목 확인
   - [ ] 추천 관광지 카드들이 표시되는지 확인

3. **현재 관광지 제외**
   - [ ] 추천 목록에 현재 관광지가 포함되지 않았는지 확인

4. **카드 클릭 동작**
   - [ ] 추천 카드 하나 클릭 → 해당 상세페이지로 이동 확인

5. **반응형 확인**
   - [ ] 브라우저 창 크기 조정 → 카드 개수가 변경되는지 확인
     - 큰 화면: 3개씩
     - 중간 화면: 2개씩
     - 작은 화면: 1개씩

6. **예외 상황**
   - [ ] 추천 관광지가 없는 경우 섹션이 숨겨지는지 확인
   - [ ] 페이지는 정상적으로 표시되는지 확인

- [ ] 예외 상황 확인 완료
- [ ] 다양한 타입/지역 확인 완료 (선택 사항)

## 문제 발견 시

다음과 같은 문제가 발견되면 개발자에게 알려주세요:

- 추천 섹션이 전혀 표시되지 않는 경우
- 현재 관광지가 추천 목록에 포함되는 경우
- 카드 클릭 시 페이지 이동이 안 되는 경우
- 레이아웃이 깨지는 경우
- 에러 메시지가 표시되는 경우
