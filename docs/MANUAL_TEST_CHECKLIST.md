# 수동 테스트 체크리스트

> 서버 실행 후 손으로 확인할 수 있는 기능 테스트 가이드
> 
> **서버 실행 명령어**: `pnpm dev`
> 
> 브라우저에서 `http://localhost:3000` 접속

---

## 1. 기본 레이아웃 및 네비게이션

### 1.1 상단 네비게이션 바 (Navbar)
- [ ] **로고 확인**
  - 왼쪽 상단에 "My Trip" 로고가 보이는지 확인
  - 로고 클릭 시 홈페이지로 이동하는지 확인

- [ ] **검색창 확인** (데스크톱)
  - 중앙에 검색창이 보이는지 확인
  - 검색창에 "관광지 검색..." 플레이스홀더가 보이는지 확인

- [ ] **메뉴 링크 확인**
  - "홈" 링크가 보이고 클릭 시 홈으로 이동하는지 확인
  - "통계" 링크가 보이고 클릭 시 통계 페이지로 이동하는지 확인
  - 로그인하지 않은 경우: "북마크" 링크가 보이지 않는지 확인
  - 로그인한 경우: "북마크" 링크가 보이는지 확인

- [ ] **로그인/로그아웃 버튼**
  - 로그인하지 않은 경우: "로그인" 버튼이 보이는지 확인
  - 로그인 버튼 클릭 시 로그인 모달이 열리는지 확인
  - 로그인한 경우: 프로필 아이콘(UserButton)이 보이는지 확인

- [ ] **스크롤 시 Navbar 고정**
  - 페이지를 아래로 스크롤해도 Navbar가 상단에 고정되어 있는지 확인

### 1.2 하단 푸터 (Footer)
- [ ] 페이지 하단에 푸터가 보이는지 확인
- [ ] 푸터에 프로젝트 정보가 표시되는지 확인

## 2. 홈페이지 - 관광지 목록 기능

### 2.1 초기 화면 로딩
- [ ] 페이지 접속 시 관광지 목록이 로드되는지 확인
- [ ] 로딩 중에는 스켈레톤 UI 또는 로딩 스피너가 보이는지 확인
- [ ] 관광지 카드가 최소 12개 이상 표시되는지 확인

### 2.2 관광지 카드 표시
- [ ] **카드 내용 확인**
  - 각 카드에 썸네일 이미지가 보이는지 확인
  - 이미지가 없는 경우 기본 플레이스홀더가 표시되는지 확인
  - 관광지명이 표시되는지 확인
  - 주소가 표시되는지 확인
  - 관광 타입 뱃지(예: "관광지", "문화시설")가 표시되는지 확인
  - 반려동물 동반 가능한 경우 🐾 아이콘과 "반려동물 동반 가능" 뱃지가 보이는지 확인

- [ ] **카드 인터랙션**
  - 카드에 마우스를 올리면 약간 확대되고 그림자가 생기는지 확인 (호버 효과)
  - 카드 클릭 시 상세페이지로 이동하는지 확인

### 2.3 필터 기능

#### 2.3.1 지역 필터
- [ ] 지역 드롭다운이 보이는지 확인
- [ ] "전체" 옵션이 기본 선택되어 있는지 확인
- [ ] 특정 지역(예: "서울", "부산") 선택 시 해당 지역 관광지만 표시되는지 확인
- [ ] URL에 `?areaCode=1` 같은 파라미터가 추가되는지 확인 (주소창 확인)

#### 2.3.2 관광 타입 필터
- [ ] 관광 타입 버튼들이 보이는지 확인 (관광지, 문화시설, 축제/행사 등)
- [ ] 특정 타입(예: "관광지") 클릭 시 해당 타입만 표시되는지 확인
- [ ] 선택된 타입 버튼이 강조 표시되는지 확인
- [ ] "전체" 버튼 클릭 시 모든 타입이 표시되는지 확인

#### 2.3.3 반려동물 필터
- [ ] "반려동물 동반 가능" 토글 스위치가 보이는지 확인
- [ ] 토글을 켜면 반려동물 동반 가능 관광지만 표시되는지 확인
- [ ] 토글을 켠 상태에서 크기 필터(소형, 중형, 대형) 버튼이 나타나는지 확인
- [ ] 크기 필터 선택 시 해당 크기만 표시되는지 확인

#### 2.3.4 정렬 옵션
- [ ] 정렬 드롭다운이 보이는지 확인
- [ ] "최신순" 선택 시 최근 수정된 관광지가 먼저 오는지 확인
- [ ] "이름순" 선택 시 가나다순으로 정렬되는지 확인

#### 2.3.5 필터 조합
- [ ] 지역 + 타입 필터를 동시에 적용해도 정상 작동하는지 확인
- [ ] 검색 + 필터를 동시에 적용해도 정상 작동하는지 확인

### 2.4 검색 기능
- [ ] **검색창 사용**
  - 검색창에 키워드(예: "경복궁") 입력
  - 엔터 키 또는 검색 버튼 클릭 시 검색 결과가 표시되는지 확인
  - 검색 결과 개수가 표시되는지 확인 (예: "검색 결과: 15개")

- [ ] **검색 결과**
  - 검색 결과가 카드 형태로 표시되는지 확인
  - 검색어와 관련 없는 결과가 나오지 않는지 확인
  - 검색 결과가 없을 때 "검색 결과가 없습니다" 메시지가 표시되는지 확인

- [ ] **검색어 초기화**
  - 검색창 오른쪽에 X 버튼이 보이는지 확인
  - X 버튼 클릭 시 검색어가 초기화되고 전체 목록이 표시되는지 확인

### 2.5 네이버 지도 연동

#### 2.5.1 지도 표시
- [ ] **데스크톱 화면**
  - 화면이 좌우로 분할되어 왼쪽에 목록, 오른쪽에 지도가 보이는지 확인
  - 지도가 정상적으로 로드되는지 확인 (한국 지도가 보여야 함)
  - 지도에 관광지 마커들이 표시되는지 확인

- [ ] **모바일 화면** (브라우저 개발자 도구로 모바일 모드 전환)
  - 상단에 "목록" / "지도" 탭이 보이는지 확인
  - "목록" 탭 클릭 시 관광지 목록이 보이는지 확인
  - "지도" 탭 클릭 시 지도가 보이는지 확인

#### 2.5.2 마커 기능
- [ ] **마커 표시**
  - 관광지 개수만큼 마커가 지도에 표시되는지 확인
  - 마커 색상이 관광 타입별로 다르게 표시되는지 확인 (예: 관광지=파란색, 문화시설=보라색)

- [ ] **마커 클릭**
  - 마커를 클릭하면 인포윈도우가 열리는지 확인
  - 인포윈도우에 관광지명, 주소, "상세보기" 버튼이 표시되는지 확인
  - "상세보기" 버튼 클릭 시 상세페이지로 이동하는지 확인

- [ ] **리스트-지도 연동**
  - 목록에서 관광지 카드를 클릭하면 지도가 해당 위치로 이동하는지 확인
  - 목록에서 관광지 카드를 클릭하면 해당 마커에 인포윈도우가 열리는지 확인
  - 목록에서 관광지 카드에 마우스를 올리면 지도에서 해당 마커가 강조되는지 확인 (호버 효과)
  - 선택된 관광지 카드가 강조 표시되는지 확인 (테두리 또는 배경색 변경)

#### 2.5.3 지도 컨트롤
- [ ] **줌 컨트롤**
  - 지도 우측 상단에 줌 인(+), 줌 아웃(-) 버튼이 보이는지 확인
  - 줌 인 버튼 클릭 시 지도가 확대되는지 확인
  - 줌 아웃 버튼 클릭 시 지도가 축소되는지 확인

- [ ] **지도 유형 변경**
  - 위성 지도 버튼(또는 지도 유형 버튼)이 보이는지 확인
  - 버튼 클릭 시 일반 지도 ↔ 위성 지도가 전환되는지 확인

- [ ] **현재 위치 버튼**
  - 현재 위치 버튼(나침반 아이콘)이 보이는지 확인
  - 버튼 클릭 시 위치 권한 요청 팝업이 나타나는지 확인
  - 권한 허용 시 지도가 현재 위치로 이동하는지 확인

### 2.6 무한 스크롤 (페이지네이션)
- [ ] 페이지를 아래로 스크롤하면 자동으로 더 많은 관광지가 로드되는지 확인
- [ ] 스크롤 중 하단에 로딩 인디케이터가 표시되는지 확인
- [ ] 새로 로드된 관광지가 지도에 마커로 추가되는지 확인
- [ ] 필터나 검색어를 변경하면 무한 스크롤이 초기화되는지 확인

### 2.7 반응형 디자인
- [ ] **데스크톱 화면** (1920px 이상)
  - 목록과 지도가 좌우로 나란히 표시되는지 확인
  - 필터가 가로로 배치되는지 확인

- [ ] **태블릿 화면** (768px ~ 1024px)
  - 레이아웃이 적절히 조정되는지 확인
  - 목록이 2열로 표시되는지 확인

- [ ] **모바일 화면** (375px ~ 767px)
  - 목록과 지도가 탭으로 전환되는지 확인
  - 목록이 1열로 표시되는지 확인
  - 필터가 세로로 배치되는지 확인
  - 검색창이 숨겨지고 메뉴만 보이는지 확인

## 3. 상세페이지 (`/places/[contentId]`)

### 3.1 페이지 접근
- [ ] 홈페이지에서 관광지 카드를 클릭하면 상세페이지로 이동하는지 확인
- [ ] URL이 `/places/125266` 형식으로 변경되는지 확인 (숫자는 실제 contentId)

### 3.2 기본 레이아웃
- [ ] **뒤로가기 버튼**
  - 상단 왼쪽에 뒤로가기 버튼(← 아이콘)이 보이는지 확인
  - 버튼에 "뒤로가기" 텍스트가 표시되는지 확인
  - 버튼 클릭 시 홈페이지로 이동하는지 확인

- [ ] **페이지 제목**
  - "관광지 상세" 제목이 표시되는지 확인

- [ ] **기본 정보 표시** (현재는 placeholder)
  - "contentId: [숫자]" 메시지가 표시되는지 확인
  - "상세 정보는 다음 단계에서 구현됩니다." 메시지가 표시되는지 확인

### 3.3 에러 처리
- [ ] 잘못된 contentId(예: `/places/abc`)로 접근 시 에러 메시지가 표시되는지 확인
- [ ] 에러 메시지에 "잘못된 관광지 ID입니다."가 표시되는지 확인

## 빠른 확인 체크리스트 (5분 버전)

핵심 기능만 빠르게 확인하고 싶을 때:

### 필수 확인 사항
1. [ ] **홈페이지 접속**: `http://localhost:3000` 접속 시 관광지 목록이 보이는가?
2. [ ] **관광지 카드**: 카드에 이미지, 이름, 주소가 표시되는가?
3. [ ] **카드 클릭**: 카드 클릭 시 상세페이지로 이동하는가?
4. [ ] **뒤로가기**: 상세페이지에서 뒤로가기 버튼 클릭 시 홈으로 돌아가는가?
5. [ ] **검색 기능**: 검색창에 키워드 입력 후 엔터 시 검색 결과가 나오는가?
6. [ ] **지도 표시**: 지도가 정상적으로 로드되고 마커가 표시되는가?
7. [ ] **마커 클릭**: 마커 클릭 시 인포윈도우가 열리는가?
8. [ ] **필터 기능**: 지역 필터 선택 시 해당 지역 관광지만 표시되는가?
9. [ ] **무한 스크롤**: 페이지를 아래로 스크롤하면 더 많은 관광지가 로드되는가?
